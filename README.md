## TOP/PKO/KOP mod: pkodev.mod.apparel
In the description of the apparel item, the name of the equipment that is fused in it appears. Modification for **[PKOdev.NET mod loader](https://pkodev.net/topic/5757-mod-loading-system-for-server-and-client-pkodevnet-mod-loader/)**.

## Modification information

- **Name:** pkodev.mod.apparel;
- **Version:** 1.0;
- **Author:** V3ct0r;
- **Type:** for client (Game.exe);
- **Supported executable .exe files:** GAME_13X_0, GAME_13X_1, GAME_13X_2, GAME_13X_3, GAME_13X_4, GAME_13X_5.
